# Agentic Architecture

Telegram -> Semantic Agent

Semantic Agent -> HALO output (broadcast to network)

Options:
Update state
New prompt to semantic agent
Create aggregator agent
Use aggregator agent
Start negotiation
Stop negotiation
Trigger actuation


Otherwise network regulates its own dispatch from checking knowledge periodically

Agents must have:
Heartbeat
Discovery
State
Functions

Agents do not have to be just in one network, can join and leave
Advocators for instance can advocate once they connect in a new household (granted permissions)
