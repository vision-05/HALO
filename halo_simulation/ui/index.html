How to use
  1. Start the API: PYTHONPATH=. uvicorn halo_simulation.server:app --reload (single worker so stream and inject share one process).
  2. Open this page from that server (e.g. http://127.0.0.1:8000) or set <meta name="halo-api-origin" content="http://127.0.0.1:8000"> if the UI is in a preview tab.
  3. Scenario CLI Human Bridge or Fused (full demo) → Run. Wait until Sim: shows a time (not —:—).
  4. Apply / Leave / Return while the stream is open. Counter / Accept need an id — after NegotiationProposal → person_cli appears, click Last nid or copy nid … from the feed.
  5. If Sim: still moves but inject says the stream is gone, the run already finished and the UI is draining the last buffered events — click Run again.
  6. Human-in-the-loop: use 1–2 Days and keep Pace (s) at ~60 so the run lasts about a minute wall-clock (inject stays open long enough to react). Set Pace to 0 for full-speed batch runs.
