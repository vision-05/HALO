How to use
  1. Start the API: PYTHONPATH=. uvicorn halo_simulation.server:app --reload (single worker so stream and inject share one process).
  2. Open this page from that server (e.g. http://127.0.0.1:8000) or set <meta name="halo-api-origin" content="http://127.0.0.1:8000"> if the UI is in a preview tab.
  3. Scenario CLI Human Bridge or Fused (full demo) → Run. Wait until Sim: shows a time (not —:—).
  4. Apply / Leave / Return while the stream is open. When NegotiationProposal → person_cli arrives, the Thermostat negotiation card shows a short green hint, fills the negotiation id, highlights Accept / Counter / Reject, and pre-fills the counter field. After you send one of those, the hint and highlights clear. Another round may prompt again.
  5. Shower proposals use minutes (feed shows shower_minutes; You card shows “shower negotiation”). With Fused, Shower / Preheat hit the tank path; after a shower, auto preheat may start if the tank is low and carbon is under the cap.
  6. If Sim: still moves but inject says the stream is gone, the run already finished and the UI is draining the last buffered events — click Run again.
  7. Human-in-the-loop: use 1–2 Days. Default Pace 0 is fastest; set Pace ~60 if you want the run to last about a minute wall-clock so inject is easier to follow (the sim thread sleeps between chunks, so higher Pace also delays inject handling).
